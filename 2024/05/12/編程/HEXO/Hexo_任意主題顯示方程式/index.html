Hexo 任意主題顯示方程式

2024-05-12
編程>HEXO
955 詞
  • hexo
  • 不正經教學
  • 採坑
  • 編程

先嘗試感覺比較多人用的 Mathjax

注意若要使用 KateX 就不要安裝有關 Mathjax 的東西! 不然可能會重複渲染出兩個公式 !

1. 先移除預設的.md編譯器

npm un hexo-renderer-marked
npm un hexo-math --save

2. 安裝要用的

npm i hexo-renderer-pandoc

注意 pandoc 需要另外到[官網](Release pandoc 3.2 · jgm/pandoc (github.com))下載安裝，安裝完要重啟電腦

3. 最後在 _config.yml 配置

math:
  enable: true
  per_page: true
  engine: mathjax

4. 缺點

pandoc 和 Mathjax 編譯都較慢，且 Mathjax 產生的方程式不能複製(至少我在支援 hexo 的版本上找不到?)

再來換嘗試 KateX

注意若要使用 KateX 就不要安裝上面任何有關 Mathjax 的東西! 不然可能會重複渲染出兩個公式 !
如果你真的遇到渲染兩個公式就重裝一次整個專案吧(我是認真的我最後也是這樣解決)

1. 先移除預設的.md編譯器

npm un hexo-renderer-marked

2. 安裝要用的

npm i hexo-renderer-markdown-it-plus --save

3. 最後在 _config.yml 配置

math:yml
  enable: true
  output: html
  per_page: true
  engine: katex

4. 缺點

KateX 有些超級超級冷門的字不支援
