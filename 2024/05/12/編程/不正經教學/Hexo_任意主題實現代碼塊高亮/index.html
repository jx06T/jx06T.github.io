Hexo 任意主題實現代碼塊高亮

2024-05-12
編程>不正經教學
1.6k 詞
  • hexo
  • javascript
  • 不正經教學
  • 原創
  • 編程

簡單版

1. 直接在 _config.yml 配置

syntax_highlighter: highlight.js
highlight:
  auto_detect: false
  line_number: true
  line_threshold: 0
  tab_replace: ''
  exclude_languages:
  wrap: true
  hljs: false
prismjs:
  preprocess: true
  line_number: true
  line_threshold: 0
  tab_replace: ''
  theme: github-dark

通常預設的 \_config.yml 就會有 syntax_highlighter 這幾項，直接把他改成這樣就好，不要讓他出現兩個 syntax_highlighter (其他也一樣)

困難版

如果你跟我一樣覺得預設的那個好醜，可以自己配置 highlight.js

1. 下載

到 Download a Custom Build - highlight.js (highlightjs.org) 選擇需要的語言後下載

2. 解壓縮

在 source 資料夾下創建一個 highlight 資料夾，將壓縮包解壓縮後全部放進去

3. 應用樣式

在根目錄創建一個 scripts 資料夾，並在裡面建立一個 Myhighlight.js內容如下

hexo.extend.filter.register('after_render:html', function (htmlContent) {
    // 添加 highlight.js 的样式和脚本
    const highlightCssLink = '<link rel="stylesheet"href="/highlight/styles/github-dark.css">';
    const highlightScript = '<script src="/highlight/highlight.js"></script>';
    
    // 在 HTML 的头部添加样式和脚本
    const modifiedHtmlContent = htmlContent.replace(/<\/head>/i,`${highlightCssLink}\n${highlightScript}\n</head>`);
    return modifiedHtmlContent;
});

4. 選主題

到 Examples - highlight.js (highlightjs.org) 挑選喜歡的樣式後，替換

const highlightCssLink = '<link rel="stylesheet"href="/highlight/styles/github-dark.css">';

中的 “/highlight/styles/github-dark.css” 中的 github-dark 為你喜歡的主題，例如：

const highlightCssLink = '<link rel="stylesheet"href="/highlight/styles/agate.css">';

就是指應用 agate.css 這個主題

5. 應用

執行來重新產生靜態檔案

hexo cl    // 清除之前建立的靜態檔案
hexo g     // 建立靜態頁面
hexo d     // 部署至 GitHub
